# Ninox – WooCommerce Integration (PHP)

##  Projektübersicht

Dieses Projekt demonstriert eine Beispiel-Integration zwischen der **Ninox API**, der **WooCommerce REST API** und einer **MySQL Datenbank** unter Verwendung von PHP.

Das Projekt wurde zu Trainings- und Lernzwecken entwickelt und dient als Demonstration von Backend-Entwicklungskenntnissen sowie API-Integration.

---

##  Funktionen

- Verbindung zur Ninox Datenbank über REST API
- Abrufen von Produktdaten aus Ninox
- Datenmapping zu WooCommerce Produktstruktur
- Erstellung neuer Produkte im WooCommerce Shop
- Verbindung zu MySQL Datenbank
- Verwendung von objektorientierter Programmierung (OOP)

---

##  Verwendete Technologien

- PHP
- Composer
- WooCommerce REST API
- Ninox API
- MySQL
- OOP Design
